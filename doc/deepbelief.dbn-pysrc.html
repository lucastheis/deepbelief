 11          """ 
 12          This class allows the construction of simpler deep belief network 
 13          architectures. It facilitates training and sampling from I{stacks} of 
 14          Boltzmann machines. However, this class does currently not support tree 
 15          structures or other more complicated arrangements. 
 16   
 17          Only after the current top most layer has been trained using L{train}() should a 
 18          new layer be added to the DBN using L{add_layer}(). After adding a new layer, the 
 19          lower layers can no longer be trained. 
 20   
 21          B{Example:} 
 22   
 23          Create and train a first layer. 
 24   
 25                          >>> dbn = DBN(RBM(10, 20)) 
 26                          >>> dbn[0].learning_rate = 1E-3 
 27                          >>> dbn[0].momentum = 0.8 
 28                          >>> dbn[0].weight_decay = 1E-2 
 29                          >>> dbn.train(data, num_epochs=50) 
 30   
 31          Afterwards, add and train a second layer. 
 32   
 33                          >>> dbn.add_layer(RBM(20, 50)) 
 34                          >>> dbn[1].learning_rate = 1E-3 
 35                          >>> dbn[1].momentum = 0.8 
 36                          >>> dbn[1].weight_decay = 1E-2 
 37                          >>> dbn.train(data, num_epochs=50) 
 38   
 39          Note that each layer has its own set of training parameters. 
 40   
 41          B{References:} 
 42                  - Hinton, G. E. and Salakhutdinov, R. (2006). I{Reducing the Dimensionality 
 43                  of Data with Neural Networks.} Science. 
 44                  - Hinton, G.E., P. Dayan, B. J. Frey and R. M. Neal (1995). I{The "wake-sleep" algorithm
